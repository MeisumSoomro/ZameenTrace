- Size: `1200 x 630`
- Format: PNG for widest social compatibility
- Safe zone: keep core text within the center `1040 x 470`

- Brand: `ZameenTrace`
- Headline: `Land, made precise.`

- Calm, technical, assured
- No crowded dashboards
- No stock photography
- Use the same neutral base and green accent from the website

## Export checklist

- Re-export `assets/social/og-preview.png` whenever the social card design changes
