## Color Scheme

```
Primary Background:    #0f1419 (Dark Charcoal)
Secondary Background:  #1a1f2e (Slightly Lighter)
Accent Color:          #3d9d8f (Teal/Emerald)
Accent Light:          #a8d5c8 (Light Teal)
Text Primary:          #ffffff (White)
Text Secondary:        #a8afa7 (Light Gray)
```

## Responsive Breakpoints

```
Desktop (1200px+)
├── 3-column grids
├── Full 96px section padding
└── 700px max-width containers

Tablet (768px - 1199px)
├── 2-column grids
├── 48px section padding
└── Adjusted font sizes

Mobile (480px - 767px)
├── 1-column stacked layout
├── 32px section padding
├── Full-width buttons
└── Smaller font sizes

Small Mobile (< 480px)
├── Single column
├── 12px section padding
├── Touch-friendly buttons
└── Minimum 44px tap targets
```

## Interactive Elements

```
Hover Effects:
├── Cards: translateY(-8px) + glow
├── Buttons: background color change + shadow
├── Links: color transition
└── Inputs: border & shadow highlight

Tab Switching:
├── Border highlight on active
├── Background color change
└── Smooth transition animation

Section Reveal:
├── fadeInUp animation
├── Staggered card reveals
└── Smooth scroll behavior
```
